Интерпретатор "Паскаль"

Служебные слова:
----------------

Program  - часть объявления имени программы (необязательно): Program test;
const    - список констант и их значений: const a=10;
var      - список переменных и их типов: var i,j:integer;
Begin    - точка входа
End      - точка выхода
Array/of - объявление массива

Типы данных
------------

2 простых типа: 
    integer и real

    -13, 34.67

сложный тип: 
    array [ integer...integer ] of <integer/real>

    Array [1..10] of Integer
 
Идентификаторы (имена переменных)
---------------------------------

Имена переменных должны запоминаться и анализироваться
1. Длина имени <= 8 символов
2. Именя могут начинаться с латинской буквы или символа "_"
3. Имена могут содержать латинские буквы, цифры и символы "_"

x, _x3_f4, e3


Операторы
---------

Математические: + - / *
Логические: < > <= >= <>

	
        *Опреатор присваивания

	x:=10;

	
	*Оператор If

	If <логическое условие> Then <оператор1> [Else <оператор2>];

        if a > b then x:=a else y:=a;

	
	*Оператор While
        
        While <логическое условие> Do <оператор>;
        
        while a>b do a:=a-1;

        
        *Оператор For (Может быть вложенным)
        
        For <переменная> := <начальное_значение> To <конечное_значение> Do <оператор>;

        for i:= 1 to 10 do x:=x+1;

        
        *READ - ввод данных с клавиатуры

        Read (список ввода)
   
        read (a, b, c);

        
        *WRITE/WRITELN - Вывод на экран

        Write (список вывода)

        write (a,b);

        << Writeln выводит данные на экран и c переходом на следующую строчку >>
          
Детали
------

*Операторы могут быть составными.
Составным оператором называется любой набор простых и составных операторов, начинающийся с Begin и оканчивающийся End.

*Комментарии обозначаются через "{" "}". Комментировать можно не только 1 строчку, но и несколько.
Мы их не учитываем, просто пропускаем с помощью get_token () или выдаем ошибку, если не нашли символ "}"

*Символ ";" является разделителем операторов. Он ставится только в том случае, если после него не идет "END"
