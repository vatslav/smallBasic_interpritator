Добро пожаловать на домашнию страницу проекта на GitHub Pages.

Вы можете скачать проект к себе на компьютер и подключиться к его разработке. Нажав на одну из папочек выше. О других способах знает google и местные мануалы, так же в вики проекта есть статья об этом. Если у вас уже все настроено то вам достаточно нажать:

$ cd your_repo_root/repo_name
$ git push git://github.com/vatslav/smallBasic_interpritator.git
$ git checkout

Т.к. автор пользуется клиентом под Windows, то способ выше не проверялся на роботоспособность
